# M1 功能范围（AI 执行用）

## 包含

| 功能 | 说明 |
|------|------|
| 企业微信登录 | OAuth 扫码/自动登录；同步组织架构（User、Department）；M1 可用 Mock Session 开发 |
| 任务 CRUD | 创建、编辑、删除（软删除）、状态变更 |
| 任务列表 | 我的任务 / 部门任务 / 全部任务（按角色过滤） |
| 进度更新 | 员工填写进度百分比、状态、阻塞说明、下一步行动 |
| 基础督办 | 定时检查截止日期；创建站内通知（企业微信推送可为 Stub） |
| 站内通知 | 被分配任务、任务即将到期、逾期、长时间未更新等；列表与已读 |

## 不包含（M1）

会议拆解、计划管理、证据上传、验收点评、AI 功能、报表。

## 角色与可见范围

| 角色 | 任务可见范围 |
|------|--------------|
| STAFF | 自己的任务 |
| MANAGER | 本部门任务 |
| DIRECTOR / CEO / ADMIN | 全部任务 |

创建任务：员工仅能给自己；主管可指派下属；超管可指派任何人。

## 验收标准（M1）

- ≥60% 员工完成登录并创建或更新过至少一条任务（可后续用真实企业微信验收）。
